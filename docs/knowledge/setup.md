# Setup Guide

This guide walks through setting up the development environment for this project locally.

## Prerequisites

- Node.js >= 18
- pnpm (recommended)

## Steps

1. Clone the repo
2. Copy `.example.env` to `.env` and edit keys
3. Run `pnpm install`
4. Start the dev server: `pnpm dev`


## Troubleshooting

- If `pnpm dev` fails, ensure `@webcontainer/api` is installed and browser supports WebContainer.
